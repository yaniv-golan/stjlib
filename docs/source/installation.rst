Installation
============

Requirements
------------
* Python 3.7 or higher
* iso639-lang>=2.4.2

Installing from PyPI
--------------------
The recommended way to install STJLib is via pip:

.. code-block:: bash

   pip install stjlib

Installing from Source
----------------------
To install from source:

.. code-block:: bash

   git clone https://github.com/yourusername/stjlib.git
   cd stjlib
   pip install -e .
